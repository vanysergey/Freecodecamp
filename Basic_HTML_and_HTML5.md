## Basic HTML and HTML5

1. Say Hello to HTML Elements
```
<h1>Hello world</h1>
```
2. Headline with the h2 Element
```
<h1>Hello World</h1>
<h2>CatPhotoApp</h2>
```
3. Inform with the Paragraph Element
```
<h1>Hello World</h1>
<h2>CatPhotoApp</h2>
<p>Hello Paragraph</p>
```
